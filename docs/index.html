CoCreate-server

A convenient chain handler allows user to chain multiple CoCreate components together. When one server is complete next one will start. The sequence goes untill all server completed. Grounded on Vanilla javascript, easily configured using HTML5 attributes and/or JavaScript API.

Install

 
npm i @cocreate/server

Or you can use cdn link: 

<script>https://cdn.cocreate.app/server/latest/CoCreate-server.min.js</script>
